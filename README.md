#### 앱 기능
- OpenWeather API를 이용하여 날씨를 보고자하는 리스트 & 선택한 도시의 날짜를 보여주는 앱
#### 기술 스택
- UIKit, RxSwift, Snapkit, Alamofire, MKMapView
#### 앱 실행 방법
- 프로젝트 zip 파일 다운
- 터미널에서 파일 위치로 이동하여 `pod install` 실행
- `.xcworkspace` 파일 실행
